### Guia de comandos para o Linux (Debian Trixie):

| **Ferramenta** | **Comando npm**                            |
| -------------- | ------------------------------------------ |
| Codex          | `npm install -g @openai/codex`             |
| Claude         | `npm install -g @anthropic-ai/claude-code` |
| Qwen-Coder     | `npm install -g @qwen-code/qwen-code`      |
| Droid          | `npm install -g droid-cli`                 |
| Copilot CLI    | `npm install -g @github/copilot`           |

| **Ferramenta**      | **Comando uv tool (uv tool update-shell)**                                              | **Update**                          |
| ------------------- | --------------------------------------------------------------------------------------- | ----------------------------------- |
| Opencode-AI         | `uv tool install opencode-ai`                                                           |                                     |
| Speckit (Atualizar) | `uv tool install specify-cli --force --from git+https://github.com/github/spec-kit.git` |                                     |
| Kimi CLI            | uv tool install --python 3.13 kimi-cli                                                  | uv tool upgrade kimi-cli --no-cache |

|**Ferramenta**|**Comando pipx**|
|---|---|
|Opencode-AI|`pipx install opencode-ai`|
|Speckit (Atualizar)|`pipx install --force git+https://github.com/github/spec-kit.git`|

| **Ferramenta** | **Comando npx (não instala)**   |
| -------------- | ------------------------------- |
| Codex          | `npx @openai/codex`             |
| Claude         | `npx @anthropic-ai/claude-code` |
| Qwen-Coder     | `npx @qwen-code/qwen-code`      |
| Droid          | `npx droid-cli`                 |
| Copilot CLI    | `npx @github/copilot`           |

|**Ferramenta**|**Comando pnpm**|
|---|---|
|Codex|`pnpm install -g @openai/codex`|
|Claude|`pnpm install -g @anthropic-ai/claude-code`|
|Qwen-Coder|`pnpm install -g @qwen-code/qwen-code`|
|Droid|`pnpm install -g droid-cli`|
|Copilot CLI|`pnpm install -g @github/copilot`|
