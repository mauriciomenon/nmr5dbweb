1. Diretório dos bancos

Defina a pasta base onde estão as versões de banco que serão analisadas.

Escolha a pasta base onde estão os bancos (.accdb/.mdb/.duckdb). Você pode digitar o caminho ou usar o botão "Navegar".

2. Escopo e filtros da busca

Especifique a tabela (opcional) e o conjunto de filtros obrigatórios.

Se vazio, o backend tenta encontrar o registro em qualquer tabela de cada banco.
Formato: COL1=VAL1,COL2=VAL2,... Valores numéricos são detectados automaticamente; use aspas para strings com vírgula.
