# Artifacts

This directory is intended to hold runtime artifacts generated by the
project, such as:

- DuckDB databases created during tests, benchmarks, or interactive use
- Benchmark result JSON files (e.g. `benchmark_results_*.json`)
- Benchmark log files (e.g. `benchmark_run*.log`)

These files are safe to delete and can be regenerated by running the
appropriate commands (benchmarks, conversions, or the web UI).

To move existing artifacts from the project root into this directory,
you can run from the repo root:

```bash
python -m tools.organize_artifacts
```

After that, the root of the repository should remain clean, while all
large/generated files live here under `artifacts/`.
